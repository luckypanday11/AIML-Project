# Iris Flower Classification Dashboard

## Overview
This project is an interactive machine learning dashboard that classifies iris flowers using multiple ML models and provides visualization and model comparison.

## Features
- Multi-model prediction (Logistic, SVM, RF, MLP)
- Interactive UI (Streamlit)
- Feature visualization
- Model comparison
- Feature importance analysis

## Tech Stack
- Python
- Scikit-learn
- Streamlit
- Matplotlib, Seaborn

## Run Locally
```bash
pip install -r requirements.txt
streamlit run app.py
